# Interactive Music Platform

## Description
Designed an interactive music platform using HTML, CSS, and JavaScript.  
The application features form handling and DOM manipulation to support newsletter sign-up and artist submission requests.  
User experience is enhanced through a responsive layout and a clean, refined UI.

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Features
- Newsletter sign-up form with validation
- Artist submission request form
- Interactive UI using JavaScript
- Responsive layout for different screen sizes

## How to Run

This project is deployed on Vercel.
